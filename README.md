**An Airbnb-style booking platform built end-to-end on the MERN stack**, covering listing management, bookings, and a production deployment pipeline.

**🎯 What it solves:** A real-world marketplace problem — connecting hosts and travelers through listing discovery, booking flows, and reliable data handling, built the way a production rental platform would need to work rather than as a toy CRUD demo.
